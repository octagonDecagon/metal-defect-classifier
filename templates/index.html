We have created a machine learning model that identifies defects in metal hot rolled metal.

Project Goals


Machine Learning Model:

Train a model that can identify metal defects with >95% accuracy. Training data will only contain microscoped images of rolled metal sheets. Use tensorflow sequential model to label model

Deployment:

Deploy a website that can allow users to upload an image in the form of PNG or JPEG, JPG and tell the user whether input is an unaccepted file type. present the output of the model in a clear way, describing each defect and its effects on the metal. It should have an aesthetically pleasant appearance. Should contain a list of all the members involved and their LinkedIn profile address Show CUCAI paper

Technologies:

The website will use Flask as its framework, HTML and Bootstrap for its appearance. Javascript will be used to add functionality to the buttons and its appearance. Tensorflow will be used to create the model and the training data will be sourced from Kaggle
